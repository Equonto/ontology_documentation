Functional Component Rolec back to ToC or Class ToC

IRI: http://spec.equonto.org/ontology/maintenance-work-management/use-case/functional-breakdown-pump-ontology#FunctionalComponentRole

The role that a material artifact bears if it is a functional part of the larger system. For example, a pump cannot operate without a motor. Therefore, the motor will play a critical component role in the pump system.
