has valuedp back to ToC or Data Property ToC

IRI: http://spec.equonto.org/ontology/maintenance-work-management/work-order-ontology#hasValue

A property allowing the assignment of concrete values to the Maintenance Work Order Record data items that require them. Allows reasoning about numbers, for instance. Aims to be compatible with IAO or similar, but specific to our information content classes for the moment. We also do not restrict the type of value to any particular literal type; whether or not there is value in specialising the data property is not yet determined.
